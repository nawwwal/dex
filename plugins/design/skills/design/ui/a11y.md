# Accessibility + Blade Score Audit

WCAG AA compliance check. Blade score check only in Blade mode.

## Mode Detection

```bash
BLADE_MODE=$(grep -q '"@razorpay/blade"' package.json 2>/dev/null && echo "yes" || echo "no")
```

---

## WCAG AA Checklist (all modes)

### Priority 1: Accessible Names (Critical)
- [ ] Images have alt text (or aria-hidden if decorative)
- [ ] Icon-only buttons have aria-label
- [ ] Form inputs have associated labels
- [ ] Links have descriptive text (not "click here")

### Priority 2: Keyboard Access (Critical)
- [ ] All interactive elements reachable by Tab
- [ ] No div/span with onClick without role + tabIndex + onKeyDown
- [ ] Escape closes dialogs/overlays
- [ ] Focus is never trapped unintentionally

### Priority 3: Focus Visibility (Critical)
- [ ] No `outline: none` without visible focus replacement
- [ ] Focus indicator meets 3:1 contrast ratio

### Priority 4: Color & Contrast
- [ ] Text contrast >= 4.5:1 (normal) or 3:1 (large)
- [ ] No information conveyed by color alone
- [ ] Touch targets >= 44x44px by default

### Priority 5: Motion & Audio (defer to /design motion)
- [ ] Animations respect `prefers-reduced-motion` — implementation patterns in `/design motion` (performance)
- [ ] Audio/sound respects `prefers-reduced-motion` — see `/design motion` (audio) and `design/ui/audio-feedback.md`
- [ ] No autoplaying media with sound

The /design motion sub-skill owns all animation and audio accessibility implementation. This checklist item is a trigger to load it.

### Touch Target Notes
- `44x44px` is the default minimum target size
- If the visible control is smaller, extend the hit area with a pseudo-element
- A constrained fallback of `40x40px` is acceptable only when `44x44px` is not feasible
- Never allow expanded hit areas to overlap neighboring interactive elements

---

## Blade Compliance Check (Blade mode only)

For each custom component found:
- Check if Blade has an equivalent component
- For hardcoded hex/spacing violations → see anti-patterns.md for the full detection list and fixes
- Target: 95%+ Blade Score
