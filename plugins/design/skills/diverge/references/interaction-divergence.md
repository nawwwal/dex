## Interaction quality gates

- Is the input method appropriate to the user's context?
- Is feedback immediate enough?
- Is the target size usable?
- Is there an undo or safe recovery?
- Does the interaction handle latency?
- Does it work with keyboard?
- Does it work with screen readers?
- Does it avoid hidden gestures for critical actions?
- Does motion explain state?
