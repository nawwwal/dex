# Motion Divergence

Use when motion, feedback, transitions, loading, or microinteraction behavior is part of the design question.

## Motion roles

- State transition
- Spatial continuity
- Progress
- Feedback
- Causality
- Attention shift
- Celebration
- Error recovery
- Loading
- Reduced-motion alternative

## Motion quality gates

- What changed?
- Where did it go?
- What caused it?
- Does the motion reduce confusion?
- Does it respect seriousness?
- Does it avoid blocking interaction?
- Does it avoid layout jank?
- Is there a reduced-motion alternative?
