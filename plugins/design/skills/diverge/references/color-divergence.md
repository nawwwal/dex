# Color Divergence

Use when color strategy is part of the design question.

## Color roles

- Background
- Surface
- Text
- Muted text
- Border
- Accent
- Primary action
- Secondary action
- Danger
- Warning
- Success
- Info
- Pending
- Disabled
- Selected
- Focus
- Chart/data colors

## Color strategies

1. Neutral product system
2. Semantic status system
3. Single-accent system
4. Brand-accent product system
5. Risk-coded system
6. Data-coded system
7. Emotional temperature system
8. Monochrome-first system
9. High-contrast accessibility-first system

## Color quality gates

- Does every color have a job?
- Is status meaning consistent?
- Is color supported by text/icon/shape?
- Does text meet contrast requirements?
- Are destructive actions unmistakable?
- Is accent overused?
- Does color help hierarchy?
- Does the UI work in grayscale?
