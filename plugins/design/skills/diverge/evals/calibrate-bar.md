# Test 4: Typography divergence

## Prompt

```md
/diverge "Diverge typography for a dense analytics dashboard."
```

## Expected

The skill should select Visual-system-level mode with Typography emphasis.

Output should include:

- Single-family product strategy
- Sans + mono strategy
- Condensed label strategy
- Numeric style
- Type scale
- Line height
- Readability and contrast
- Why decorative display fonts are likely wrong

## Pass criteria

- It names product register unless the prompt adds brand context.
- It treats typography as hierarchy/readability infrastructure.
- It includes numeric/tabular style.
- It includes accessibility and localization considerations.
- It does not propose expressive display typography as the default for dense analytics.
