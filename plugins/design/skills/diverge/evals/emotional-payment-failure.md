# Test 6: Emotional design

## Prompt

```md
/diverge "Diverge emotional design for a payment failure recovery flow."
```

## Expected

The skill should select Mixed or UX-level with Emotional Design emphasis.

Pass criteria:

- Target emotion includes safety, relief, and confidence.
- No jokes.
- Clear copy.
- Data safety reassurance.
- Recovery-first hierarchy.
- Gentle motion.
- Persuasive but ethical retry prompt.
- Error states explain what happened, why, what now, and whether data is safe.
- Handoff includes states, events, analytics, accessibility, and QA.
