# Brand And Reference Brief

Use this as source material for a Reveal.js deck.

Brand:

- Razorpay primary colors: `#305eff`, `#5278ff`, `#6ed00b`, `#c1ff84`.
- Razorpay secondary colors: `#00be5f`, `#389494`, `#387594`.
- Razorpay tertiary colors: `#ff8a80`, `#826dff`, `#fbec51`, `#7dd5e9`.
- Use Tasa Orbiter Display or a documented fallback for headlines.
- Use Inter for body text.
- Use small mono labels only for annotations.
- Keep the logo small, clear, and away from the main claim.

Reference mechanics:

- Black-stage workshop slides use full-bleed black, white text, chunky chips, symbol tiles, and mono annotations.
- Pale-mat exhibit slides use a full-bleed pale surface with a large dark exhibit panel and a caption claim.
- Event atmosphere slides use huge uppercase headlines, pale image fields, and footer rules.

Non-negotiable:

- Do not switch type systems slide-by-slide.
- Do not render a centered slide container inside browser whitespace.
- Body text must meet at least 4.5:1 contrast.
- Large display text must meet at least 3:1 contrast.
