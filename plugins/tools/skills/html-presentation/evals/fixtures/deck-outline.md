# Agent Marketplace Installation Review

Audience: product, design, and engineering reviewers.

Decision needed: approve the Phase 2 install flow only if users can recover from connector setup failure.

Source material:

- Discovery is not the hard part; recovery from failed setup is the gating risk.
- The deck should show the primary install path, failure handling, and support handoff.
- Reviewers need to inspect real UI states, not only read a process list.
- Keep optional backend/API details in an appendix.

Required slides:

1. Decision frame.
2. Current failure.
3. Proposed install flow.
4. Recovery states.
5. Open questions.
6. Appendix with API assumptions.
