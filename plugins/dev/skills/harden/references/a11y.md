# Accessibility Audit

WCAG AA floor for product interfaces.

## Route Out First

- Blade-specific compliance, Blade Score, or component replacement: route to `blade`.
- In Blade projects, motion timing, reduced-motion implementation, hover/focus/tap behavior, or component replacement: route to `blade` first. Use generic motion guidance only after Blade MCP proves no Blade-native path.
- Sound or audio feedback design: route to `create-sound` or the available sound skill.

## Checklist

### Accessible Names
- Images have alt text, or `aria-hidden` if decorative.
- Icon-only buttons have `aria-label`.
- Form inputs have associated labels.
- Form groups use `fieldset`/`legend` or equivalent structure when one label governs multiple controls.
- Links use descriptive text, not "click here".

### Keyboard Access
- Interactive elements are reachable by Tab.
- Clickable `div`/`span` elements have role, tabIndex, and keyboard handling, or are replaced with semantic controls.
- State-changing actions use `button`; navigation uses `a[href]`.
- Escape closes dialogs and overlays.
- Focus is never trapped unintentionally.

### Focus Visibility
- No `outline: none` without a visible replacement.
- Focus indicator has at least 3:1 contrast.
- Focus order matches visual order.

### Color And Contrast
- Normal text contrast is at least 4.5:1.
- Large text and meaningful icons are at least 3:1.
- Color is not the only carrier of meaning.

### Touch Targets
- Default target size is at least 44x44px.
- A constrained 40x40px fallback is acceptable only when 44x44px is not feasible.
- Expanded hit areas do not overlap neighboring controls.

### Media
- No autoplaying media with sound.
- Audio controls are visible and keyboard accessible.

### Semantic Structure
- Headings describe document structure, not visual size.
- Landmarks such as `main`, `nav`, `header`, and `footer` are used only for their real page roles.
- Native form submission, validation, and autocomplete behavior are preserved unless the replacement is fully accessible.
