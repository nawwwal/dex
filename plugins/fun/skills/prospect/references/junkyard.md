# The junkyard

A running collection of invented two-player word games and how they died. Broken games are kept on purpose — they are funnier and more instructive than most working games, and the collection is the point of `prospect`. Append each new invention with its one-line rules and its cause of death (or, rarely, "KEEPER").

Format:

```
## <Game name>
Rules (1 line): <objective + core move>
Verdict: <KEEPER | degenerate | unplayable | flat>
Cause of death: <the specific reason it broke, if it broke>
```

---

## Echo Ladder (seed example)
Rules (1 line): take turns saying a word that shares exactly one letter with the previous word in the same position; first to fail to find one loses.
Verdict: degenerate
Cause of death: "exactly one letter in the same position" is so loose that valid moves are nearly infinite — the game effectively never ends and has no tension. A near-miss; tightening to "shares its first letter and is longer" might rescue it. Logged for the morgue and for the rescue idea.
