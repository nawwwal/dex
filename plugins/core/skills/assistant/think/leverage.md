# Find Highest Leverage Focus

Identifies where concentrated effort will compound most.

## Step 1: Load Goals and Status
- Read memory/goals.md — current OKRs, promotion criteria, priorities
- Read career/case.md — evidence gaps by competency
- QMD search `work/` for active projects (`qmd_vsearch "active project status"`)
- Read memory/patterns.md — repeated patterns (skills you've already developed)

## Step 2: Map Current Investment
For each active project/skill: estimate hours invested vs. impact delivered.

## Step 3: Find Leverage Points
Look for:
1. **Competency threshold opportunities** — one more evidence entry closes a gap
2. **Relationship leverage** — one conversation changes multiple outcomes
3. **Skill compounding** — learning X makes you better at A, B, and C
4. **High-signal low-effort** — what takes 2 hours but looks like 2 weeks of work?

## Step 4: Output
3-5 specific leverage points with vault evidence:
```
**Leverage Point 1: [title]**
Evidence: [what the vault says about this opportunity]
Effort: [estimated time]
Impact: [what changes]
Action: [specific next step]
```
