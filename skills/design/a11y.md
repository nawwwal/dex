# Accessibility + Blade Score Audit

WCAG AA compliance check + Blade score verification.

## WCAG AA Checklist

### Priority 1: Accessible Names (Critical)
- [ ] Images have alt text (or aria-hidden if decorative)
- [ ] Icon-only buttons have aria-label
- [ ] Form inputs have associated labels
- [ ] Links have descriptive text (not "click here")

### Priority 2: Keyboard Access (Critical)
- [ ] All interactive elements reachable by Tab
- [ ] No div/span with onClick without role + tabIndex + onKeyDown
- [ ] Escape closes dialogs/overlays
- [ ] Focus is never trapped unintentionally

### Priority 3: Focus Visibility (Critical)
- [ ] No `outline: none` without visible focus replacement
- [ ] Focus indicator meets 3:1 contrast ratio

### Priority 4: Color & Contrast
- [ ] Text contrast >= 4.5:1 (normal) or 3:1 (large)
- [ ] No information conveyed by color alone
- [ ] Touch targets >= 44x44px

### Priority 5: Motion
- [ ] Animations respect `prefers-reduced-motion`
- [ ] No autoplaying media with sound
