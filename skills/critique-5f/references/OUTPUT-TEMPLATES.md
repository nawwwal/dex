## Usage Instructions

1. **Choose template based on user's view preference** (from Phase 1, Step 6)
2. **Fill in placeholders** using outputs from Agents 1, 2, and 3
3. **Keep Simple View under 1000 words**
4. **Keep Advanced View under 2000 words**
5. **Always include prioritization framework** in both views
