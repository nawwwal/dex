This file is automatically loaded before every 5F design review to provide learned context.

2. **Context questions** - Targeted questions when skill is uncertain

4. **Weekly analysis** - Automated pattern detection every 7 days

**You control learning by:**
- Providing optional feedback after reviews
- Answering context questions (or skipping them)
- Editing this file directly anytime
- Running `/context-5f` commands
